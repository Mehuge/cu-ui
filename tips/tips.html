Welcome to Camelot Unchained's
Extended Alpha.
Please review the following tips:
Press V to create new spells/abilities
Press B to open spellbook to delete spells/abilities
Type '/hideui perfhud' to hide the statistics window
Type '/suicide' to quickly spawn in a new location
To help increase performance on older systems, type 'shadowMaxDist 0', hold Shift, and press Enter.
To run the game in full screen at high resolution, hold Alt while clicking the 'Play' button on the launcher and enter 'windowWidth=1920;windowHeight=1080'.
For other very useful information, please click the 'Alpha Manual' link on the game patcher.
